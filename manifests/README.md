# Fit manifests

`librispeech_audio_fit_128.jsonl` is generated by
`scripts/modal_fit_mixed_lens.py --stage-only` from the pinned LibriSpeech
revision. It contains metadata and hashes for the exact 128 utterances used by
the mixed WikiText/audio Jacobian-lens fit. Raw audio stays on the private
Modal volume and is not committed.
